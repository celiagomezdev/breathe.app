# breathe.app

A mobile-first map of smoke-free bars in Berlin.

> ### 🚀 Hackathon & R&D Learning Sandbox
>
> **This repository is dedicated to internal research, development, and tool-evaluation during project transition phases and company Hackathon time.**
>
> - **Core Objective:** The goal of this project is to rigorously evaluate **Claude Code (CLI)**, advanced LLM prompting strategies, and AI agentic workflows in a modern web ecosystem.
> - **Technical Focus:** Benchmarking AI velocity while building a mobile-first framework architecture using React Router v7, TailwindCSS, and serverless data layers.
> - **Internal Value:** Insights gained regarding prompt engineering efficiency, codebase context management, and automated file generation will be used to optimize internal team development workflows and accelerate future production project lifecycles.
